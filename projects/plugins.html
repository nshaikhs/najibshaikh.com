What it is

Four Claude Code skill packs that install domain expertise directly into your AI agent. Each pack makes Claude Code perform like a specialist, not a generalist giving generic advice, but an expert applying real frameworks and methodologies.

These aren't prompt templates. They're structured skill files that integrate with Claude Code's native skill architecture and MCP server ecosystem. Install a pack and your agent immediately understands the domain at a practitioner level.

The packs

Marketing Suite

Cold email sequences using the Curiosity Ladder methodology. SEO content strategy with E-E-A-T framework compliance. Ad creative generation. Content calendar planning. Outreach personalization at scale.

Design Toolkit

UI/UX design systems with component libraries. Brand identity development from brief to guidelines. Design token architecture for consistent theming. Responsive layout patterns. Accessibility-first component design.

Workflow Automation

SEO Engine

What makes it different

Most AI tools give you access to a general-purpose model and call it "AI-powered." These packs bake in domain expertise. The Marketing Suite doesn't just "write emails." It applies the Curiosity Ladder methodology to structure subject lines and opening hooks. The SEO Engine doesn't just "optimize content." It runs E-E-A-T framework analysis and generates schema markup that actually validates.

The difference is operator-built vs. developer-built. I use these packs daily across my own work. Every skill was refined through real usage: running actual outreach campaigns, building actual websites, auditing actual SEO performance. The feedback loop is tight because I'm both the builder and the user.

How it works

Install

Download the pack from Gumroad. Drop the skill files into your Claude Code skills directory. The skills register automatically. No configuration needed.

Activate

Claude Code detects the installed skills and makes them available in context. When you ask a domain-relevant question, the right skill activates with the right framework and methodology.

Execute

The skills produce structured, actionable output, not generic advice. A cold email skill generates a complete sequence with subject lines, body copy, follow-up timing, and A/B test variants.

Integrate

Skills work with Claude Code's MCP servers. The CRM skill connects to your actual CRM. The SEO skill audits your actual site. The email skill drafts in your actual email tool. Real integrations, not simulations.
